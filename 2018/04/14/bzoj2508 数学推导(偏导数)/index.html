bzoj2508 数学推导(偏导数)

发表于 2018-04-14 | 分类于 bzoj

###题目大意： 求一个点使得它到平面上所有直线距离平方和最小。 你需要实现以下3种操作：

  1.   平面上加入一条直线；

  2.   删除一条已加入的直线；

  3.   求一个点到平面上所有直线距离平方和最小，你需要输出这个最小值。

###输入格式： 第1行包含一个整数N，表示了操作数目。接下来N行操作属于下列3种格式之一： 格式1: 0 x1 y1 x2 y2 。插入直线操作，插入一条过(x1, y1), (x2, y2)的直线，保证两点不重合，坐标为实数（最多两位小数）并且绝对值不超过100。 格式2: 1 k 。删除直线操作，删除第i次插入操作所插入的直线，保证已经被删除的直线不会再被删除（即任意2次操作k值均不同），并且k不大于之前插入操作的次数。 格式3: 2 。查询操作，查询所要求的最小值。 ###输出格式： 输出行数等于查询操作的次数，每行输出每次查询操作所要求的最小值，保留两位小数. ###样例输入： 10 0 0.0 0.0 1.0 0.0 2 0 0 1 1 1 2 0 0 2 1 2 2 1 2 2 1 3 2 ###样例输出： 0.00 0.50 2.00 2.00 0.00 ###分析： (参考自claris的博客) 先把题目要求的式子给列出来，利用点到直线的距离公式. 这实际上就是要求二元函数$f(x,y)$的最小值.可以利用偏导数等于0来用$ai,bi,ci$分别表示$x,y$,但是用这种方法解到一半就会崩溃,其实不必解出$x,y$,只需要用$x$表示出$y$,再带入到原式子中求最值即可. 具体推导过程如下： 其实利用$y$的偏导数等于$0$也可以求出$y$的表达式. 发现$y$是一次函数的形式，将那一大坨式子换元，能够起化简式子的作用. 最后就是带入了: $x$取对称轴时，二次函数取最小值.（仅仅是这样吗？） 在初三学抛物线时经常会在这里出错，要特判$U = 0$的情况！$U = 0$其实就是求一次函数的最值，答案是$W$. 最后维护一大坨前缀和就可以了. 几个坑点： 1.查询的时候可能没有直线,输出$0$即可. 2.$A、B$的分母可能为0，那么$A、B$也为0. 3.最后的答案绝对值如果＜eps,直接输出$0.00$. 4.注意插入直线两个点横坐标相等的情况. 总结： 1.初中学过的数学公式要记全！比如点到直线的距离公式、抛物线的对称轴等等. 2.发现可以用公式表示出答案后一定要写下完整的公式！我一开始没有考虑到有$Σ$所以推偏导发现$x,y$解不出来. 3.每一步数学推导都要写在草稿纸上，一旦发现可以套模型（一次函数、二次函数），果断换元化简式子！ 4.如果题目给直线的方式是给定两个点，一定要注意两个点横坐标相等的情况! 5.小技巧：如果最后的答案有可能<eps,那么输出0即可. 最后是我丑陋的代码：
