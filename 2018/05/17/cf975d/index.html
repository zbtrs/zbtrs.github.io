Codeforces 975 D. Valhalla Siege

发表于 2018-05-17 | 分类于 计划

英文题面

翻译

  给定直线$y = ax + b$上的若干个点的x,以及这些点的横向速度以及纵向速度. 求这些点会相交多少次.

提示

  两个点相遇的条件是什么?

代码

分析

  两个点相遇的条件是：在同一时刻，它们的横坐标相同，纵坐标相同，那么我们可以针对横坐标和纵坐标的时间列一个方程，解得：$v_{ay} - av_{ax} = v_{by} - av_{bx}$. 利用map存$v_y - av_x$的值有多少个，每次在map中查找个数即可.
  因为列的方程的分母可能为0，需要再建一个map存所有的直线(由横速度与纵速度构成)，来保证不计数平行的直线.

总结

  对问题的条件做进一步的推导与转化. 若要计数同类元素，记录值域信息！
