中文题面

Description
神犇SJY虐完HEOI之后给傻×LYD出了一题：SHY是T国的公主，平时的一大爱好是作诗。由于时间紧迫，SHY作完诗
之后还要虐OI，于是SHY找来一篇长度为N的文章，阅读M次，每次只阅读其中连续的一段[l,r]，从这一段中选出一
些汉字构成诗。因为SHY喜欢对偶，所以SHY规定最后选出的每个汉字都必须在[l,r]里出现了正偶数次。而且SHY认
为选出的汉字的种类数（两个一样的汉字称为同一种）越多越好（为了拿到更多的素材！）。于是SHY请LYD安排选
法。LYD这种傻×当然不会了，于是向你请教……问题简述：N个数，M组询问，每次问[l,r]中有多少个数出现正偶
数次。
Input
输入第一行三个整数n、c以及m。表示文章字数、汉字的种类数、要选择M次。第二行有n个整数，每个数Ai在[1, c
]间，代表一个编码为Ai的汉字。接下来m行每行两个整数l和r，设上一个询问的答案为ans(第一个询问时ans=0)，
令L=(l+ans)mod n+1, R=(r+ans)mod n+1，若L>R，交换L和R，则本次询问为[L,R]。
Output
输出共m行，每行一个整数，第i个数表示SHY第i次能选出的汉字的最多种类数。

Sample Input
5 3 5

1 2 2 3 1

0 4

1 2

2 2

2 3

3 5
Sample Output
2

0

0

0

1
HINT
对于100%的数据，1<=n,c,m<=10^5
Source
By lydrainbowcat

代码

分析

  如果做过蒲公英那道题，很容易就能想到这道题可以用分块来做. 记录一个数组$f_{i,j}$表示第i个块到第j个块内出现次数为偶数的数有多少个. 可以通过$O(sqrt(n)n)$预处理得到.然后用一个vector存储每个数值在哪些位置出现了.
  对于查询操作，先查整块内的值：若l属于块p,r属于块q,$ans = f_{p + 1,q - 1}$,然后暴力统计不在整块内的数在给定区间内是否出现了偶数次，利用vector对区间的左右端点进行lower_bound和upper_bound运算. 为了避免重复统计，可以利用vis数组标记已经统计过的值，因为值域很小，不需要离散化. 查询操作很多，利用时间戳来替代vis的清空. 这样做会出现一个问题：可能有的数在整块内出现了偶数次，在区间[l,r]内出现了奇数次，那么这个数对应的答案就要减掉;或者在整块内出现了奇数次，在[l,r]内出现了偶数次，这个数对应的答案就要加上. 对这两种情况进行特殊判断即可.
  需要注意的是题目要求的出现偶数次是正偶数次，统计的时候特判出现0次的情况!
