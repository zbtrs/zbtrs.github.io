Codeforces 980 D.Perfect Groups

发表于 2018-05-13 | 分类于 计划

英文题面

翻译

有一个问题：从给定的将给定的集合划分成尽量少的子集，使得每个子集内的每一对数的乘积是完全平方数.
现在给你一个数组A,让你求有多少个子数组(元素相邻)使得问题的答案为k,输出k为1~n的所有答案.

提示

  由完全平方数能想到什么？

代码

分析

  遇到完全平方数，先将所有数分解质因数. 若一个数为完全平方数，则它的每一个质因子的次数都是偶数.若两个数的乘积是完全平方数，那么这两个数对应的质因子的次数要么同时是奇数，要么同时是偶数. 因为要求乘积的质因子的次数为偶数，若当前这个数的某个质因子的次数已经是偶数了，我们就不需要考虑这个质因子的影响,只需要考虑它的次数为奇数的质因子的影响.
  将每个数换一种表示方法，比如180 = 2*2*3*3*5,把次数为偶数的质因子去掉，就变成了5. 对于一个已经确定的集合，该集合最少被分成的子集数=有多少种不同的表达方式. 将所有数的表达方式都存到一个数组中，离散化一下. 因为n只有5000，可以枚举连续段的左右端点,固定左端点，枚举右端点，同时统计答案.
  需要考虑负数和0的情况. 负数的表示方法一定带一个负号，若连续的一段数只包含0，则最少划分成一个子集，若包含其他的数，0可以忽略！

经验教训

  一开始想着把所有的表示方法状压一下，看来还是too young too simple. 感觉可以把这个表示方法理解成hash函数，将状压转换成hash,效率高很多.
