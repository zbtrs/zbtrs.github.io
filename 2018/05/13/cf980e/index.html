Codeforces 980 E.The Number Games

发表于 2018-05-13 | 分类于 计划 |

英文题面

翻译

有一棵n个节点的树，第i个节点的权值为$2^i$,需要删掉K个点，使得剩下的点连通，并且保留下来的点的权值和尽量大，求删哪些点？

提示

  $2^i$有什么性质？

代码

分析

  非常巧妙的一道题. 我一开始的想法是把所有度数为1的点放到优先队列中，取出权值最小的点删掉，继续加点删点，直到正好删去k个点.感觉似乎没问题，实际上当有4个点的权值分别为1,6,4,5时，要删掉3个点，若1的度数不为1，那么就有可能删掉4,5,6,不是最优的.于是我又想到了poj color the tree,每次删点要找最外围的点嘛，那么能不能找到一种依赖关系呢？ 显然不行，因为你可以以多种方式删掉一个点。
  删点不行，能不能考虑保留哪些点呢？上述做法说明了我们不能优先确定顺序，而应该确定保留哪些点. 我当时忽视了一个性质，导致一直想不出来：$2^i = 2^{i - 1} + 2^{i - 2} + …… + 1$,也就是说选i号点比选所有的编号小于i的点都要优. 那么算法就出来了，按照编号从大往小选,选的那些点一定会构成一个连通块，若当前考虑的是第i个点，如果它与连通块之间的路径上有cnt个点，若cntt(已经选的点) + cnt + 1 > n - k,则不选，否则选，将路径上的所有点打上标记,利用倍增找到深度最浅的没有被标记的点. 以最后一个点为根建树.

经验教训

  这类贪心问题都要从产生收益的一方取考虑啊！千万不要舍本逐末取考虑顺序条件什么的.
