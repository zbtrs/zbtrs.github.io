bzoj2844 albus就是要第一个出场

发表于 2018-05-09 | 分类于 计划

###中文题面 Description 已知一个长度为n的正整数序列A（下标从1开始）， 令 S = { x | 1 <= x <= n }, S 的幂集2^S定义为S 所有子 集构成的集合。定义映射 f : 2^S -> Zf(空集) = 0f(T) = XOR A[t] , 对于一切t属于T现在albus把2^S中每个集 合的f值计算出来， 从小到大排成一行， 记为序列B（下标从1开始）。 给定一个数， 那么这个数在序列B中第1 次出现时的下标是多少呢？ Input 第一行一个数n, 为序列A的长度。接下来一行n个数， 为序列A， 用空格隔开。最后一个数Q， 为给定的数.

Output 共一行， 一个整数， 为Q在序列B中第一次出现时的下标模10086的值. Sample Input 3

1 2 3

1 Sample Output 3

样例解释：

N = 3, A = [1 2 3]

S = {1, 2, 3}

2^S = {空, {1}, {2}, {3}, {1, 2}, {1, 3}, {2, 3}, {1, 2, 3}}

f(空) = 0

f({1}) = 1

f({2}) = 2

f({3}) = 3

f({1, 2}) = 1 xor 2 = 3

f({1, 3}) = 1 xor 3 = 2

f({2, 3}) = 2 xor 3 = 1

f({1, 2, 3}) = 0

所以

B = [0, 0, 1, 1, 2, 2, 3, 3]

HINT 数据范围：

1 <= N <= 10,0000

其他所有输入均不超过10^9

###分析   根据hdu3949,我们可以通过将Q分解成二进制，从而计算Q是第几小的. 具体方法是：维护一个变量cnt. 若从大到小第i个线性基存在于Q中，那么$cnt += 2^i$. 但是这道题的计数并没有这么简单,异或空间的异或值可能有重复的.   一定能从$a_1,a_2,......,a_t$中找出t个线性无关的整数，构成异或空间的一个基。不在基底的整数有n-t个，从这n-t个整数中选出若干个，显然有$2^{n-t}$种方案. 对于任意一种选法，设选取的整数异或起来得到x,把基底能表示出的$2^t$个互不相同的整数与x分别异或，又得到$2^t$个整数,这些整数互不相同. 因此可以得到一个结论：不去重异或集合就是去重异或集合中的$2^t$个整数个重复$2^{n-t}$次形成的 根据这个结论，最后的答案就是$cnt * 2^{n-t} + 1$. ###一些容易掉进去的坑   1.一定要注意负数情况啊!!!!! 将所有的负数取相反数变成正数即可.   2.用高斯消元法(代码1)，要求出所有基的最高位是哪一位，还要排序. 一定要记住：高斯消元法求出来的第i个基的最高位不一定是n-i+1,推荐使用代码2的方法.
