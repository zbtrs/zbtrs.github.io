Codeforces 311 B. Cats Transport

发表于 2018-05-26 | 分类于 算法竞赛进阶指南

###题目大意    p个人,m只猫,n座山，告诉每座山距离前一座山的距离，和每只猫在哪座山上，等待时间. 所有人一开始都在一号山，每单位时间挪动距离为1，求如何安排这些人的出发时间使得所有的猫的等待时间总和最少？ ###代码

###分析   首先要知道猫在什么条件下才能被接到. 对于每只猫，设$A_i = T_i - ΣD_j(1≤j≤H_i)$.只有当人的出发时间在$A_i$之后时，这只猫才能被接到，等待时间为$t - A_i$,t是出发时间.   将所有猫按照A从小到大排好序后，就能够分配人去接猫了. 每个人接的猫对应的A一定是连续的一段. 因为要考虑有多少个人，接了多少只猫，因此状态定义为：$f_{i,j}表示前i个人，带走第k+1~j$的费用，枚举k,即可得到状态转移方程：$f_{i,j} = min{f_{i - 1,k} + A_j *(j - k) - (S_j - S_k)}$,其中S是A排好序后的前缀和. 斜率优化就能过. ###坑点   一定要注意初始化！
