算法竞赛进阶指南第四章

发表于 2018-05-12 | 分类于 算法竞赛进阶指南 |

前言

  花了5天的时间把这一章学完了. 发现有好多数学定理我以前都不会证明，现在学了一遍，算是建立了比较完整的知识体系吧.

1.余数与除法分块

  遇到余数，要么是变成同余式处理，要么是将余数化为余数表达式. 即k mod i = k - (k / i) * i. 或者是r = k - pi. 注意到第一个式子有k / i下取整. 我们可以利用商是由若干段连续的数字拼成的这一事实来优化算法. 当i属于 [x, k /(k / x)]时，[k / i]的值都相等.

2.积性函数的求值方法.

  对于积性函数$f(x)$，通常将$x$分解为$p_1^{q_1}p_2^{q_2}…p_n^{q_n}$,分别求出$f(p_i^{q_i})$的值，最后再乘起来就是$f(x)$的值了，这利用了一个数的不同质因子间互质这一定理.

3.欧拉函数的若干性质

  1.对于所有的大于1的n,1~n中与n互质的数的和为$n*φ(n) / 2$.
  2.若a,b互质，则$φ(ab) = φ(a)φ(b)$. （欧拉函数是积性函数）
  3.若$p | n$且$p^2 | n$,则$φ(n) = φ(n / p) * p$.
  4.若第3条中的$p^2$不整除$n$,则$φ(n) = φ(n / p) * (p - 1)$.
  5.若d为n的约数，那么$Σφ(d) = n$.
  因为$gcd(n,x) = gcd(n,n-x)$,即与n不互质的数成对出现，平均值为$n/2$，可以证明性质1，利用性质1可以求与一个数不互质的数的和.
  利用第2,3,4条性质，可以在线性筛中求出$φ(n)$的值,性质二的证明可以利用上面提到的积性函数的求值方法：将a,b分解质因数. 性质3可以通过公式展开直接证明. 性质4则利用性质2证明.
  对于第5条性质,令$g(n) = Σφ(d)$,证明$g(n)$是积性函数，利用积性函数的求值方法即可证明. 如何证明$g(n)$是积性函数？只需要证明$g(n)g(m) = g(nm)$即可，其中n,m互质.

4.费马小定理的证明

  费马小定理：若p是质数，对于任意整数a,有$a^p = a （mod p)$
  由费马小定理可以导出欧拉定理$a^{φ(p)} = 1(mod p)$,由于$p$是质数，
所以$φ(p) = p - 1$,证明了欧拉定理即可证明费马小定理.
  利用简化剩余系来证明. 若$\{a_1,a_2,…,a_{φ(n)}\}$与$\{aa_1,aa_2,…,aa_{φ(n)}\}$都能表示模n下的简化剩余系，那么$a^{φ(n)}a_1a_2…a_{φ(n)} = (aa_1)(aa_2)……(aa_{φ(n)}) = a_1a_2…a_{φ(n)}(modn)$,也就是将上述两个集合分别乘起来,因为a与n互质，所以a可以消掉，就得到了上面的定理，同时也证明了欧拉定理. 这一步利用了一个关键的性质：模n意义下，与n互质的数不影响整除性，故a可以消掉.
  上面的结论都是在这两个集合是等价的前提下推导出来的，下面来证明这两个集合是等价的. 我们只需要两个集合的元素是一一对应的即可. 对于n原来的简化剩余系，若有$a_i,a_j$,$aa_i = aa_j(modn)$,那么a可以消掉，得到$a_i = a_j(modn)$. 当$a_i ≠a_j$ 时，$aa_i,aa_j$一定对应着不同的同余类. $a_i对应aa_i$,两个集合大小相等，那么这两个集合就是等价的.

5.裴蜀定理的证明

  先证明边界情况：b = 0时,x = 1,y = 0. 再对欧几里得算法的递归过程应用数学归纳法. 若$bx + (amodb)y = gcd(b,amodb)$利用余数的展开式，对式子进行变形，令$x’ = y,y’ = x - (a/b)y$,因为$gcd(b,amodb) = gcd(a,b)$,所以$ax’ + by’ = gcd(a,b)$,正好就是欧几里得算法往上一层的情况.

6.乘法逆元与中国剩余定理

  一定要注意：乘法逆元只有保证在模数为大质数的情况下才一定会有.若p为小质数，x是p的倍数，那么x不存在模p意义下的逆元.
  中国剩余定理类似拉格朗日插值法，都是利用特殊的点值，只有当$x = x_i$时，$y = y_i$. 当$x = x_k (k≠i)$时，$y = 0$,因此可以将答案写成$Σa_iM_it_i(modm_i)$.
中国剩余定理求出来的解一定要不断地加mod使之变成非负数！
  若模数不互质，可以用增加模数的lcm法，或者直接合并两个方程.

7.矩阵快速幂

  如果一类问题满足以下特点：
  1.可以抽象出一个长度为n的一维向量，该向量在每个单位时间发生一次变化.
  2.变化的形式是一个线性递推(只能乘上or加上一个数).
  3.该递推式在每个时间可能作用于不同的数据上，但本身保持不变.
  4.向量变化时间(递推的轮数)很长，但向量长度n不大.
  即可使用矩阵快速幂优化递推.
  设计转移矩阵也有很多技巧. 若状态矩阵的第i维对下一层状态矩阵的第j维有影响，那么转移矩阵的$a_{i,j} = ?$,?根据题目来定. 状态矩阵是一个横着的矩阵，其列数与转移矩阵的行数相等.
  若递推式具有周期性(在同一个周期内的不同时间递推式不同)，周期很小，那么可以构造周期长度个矩阵.

8.高斯消元and线性基

  高斯消元可以用来解线性方程组，异或方程组，模线性方程组(我目前遇到的)，线性基主要是用来处理异或问题的.
  高斯消元有四类不同的写法:
  1.有且仅有一组解的线性方程组. （模）
  2.贪心+线性方程组.
  3.异或方程组.
  4.有多种解的情况的线性方程组.（模）
  第一种怎么写都可以. 第二种是类似于线性基一样的写法，不交换行的位置，每次消掉能找到的最高位. 第三种就是每次找最大的数，换到第i行. 消掉其它的i的最高位为1的数. 第四种要维护当前消到了第几行，消到了第几列. 优先保证第i行被消掉，而不是贪心中的保证第j列被消掉，以下是四种高斯消元的代码：

9.概率/期望/01分数规划/博弈

  期望dp要倒推，概率dp要正推. 期望dp如果要正推就要先求出到达每个点的概率,转移的代价要乘上概率.
  01分数规划用二分.
  sg函数用来求解有向图游戏. 在应用sg函数之前，必须先将失败条件转化为一方不能移动.,可以通过微调边界来实现.
  mex与xor的区别：mex对应的是每一种决策所转移到的状态的sg值的集合,xor是一种决策将当前任务分成的若干个子任务的sg值的运算.
