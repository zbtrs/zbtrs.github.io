Codeforces 876 E. National Property

发表于 2018-05-25 | 分类于 计划

###题目大意
规定大写字母的字典序小于小写字母. 现在将所有的字母都表示成数字,可以将若干个字母变成大写字母，替换所有单词中的该字母. 问能不能将这些单词的字典序变成单调上升的? 如果能，输出变哪些字母.

###提示
  一个字母对应两种状态，能想到什么？

###分析
  打比赛时非常遗憾的一道题. 一直想的是贪心or dp,就是没往2-sat上面想. 其实2-sat也很容易想到. 因为每个字母只有两种状态. 考虑有哪些限制.
  1.若第i行与第i+1行相邻的两个字母x,y, x < y，那么这两个字母是合法的，x不变y就不能变，y变了x就必须变，所以连边x –> y, y’ –> x’.
  2.若x > y,那么x必须变，y不能变. 连边x –> x’,y’ –> y.
  剩下的就是2-sat模型了. 最后要输出方案，拓扑排序一下即可.

###教训
  1.字母不要一列一列地存，要一行一行地存，不然可能会出现两个字母在行上不连续出现的情况.
  2.要检验一种不合法的特殊情况：第i个和第j个单词的前len位一样(len是两个单词长度的较小值)，但是它们的长度不一样. 这样无论怎么变都不可能使单词的字典序有序.
