poj2411 Mondriaan's Dream

发表于 2018-05-21 | 分类于 算法竞赛进阶指南 |

英文题面

Description

Squares and rectangles fascinated the famous Dutch painter Piet Mondriaan. One night, after producing the drawings in his ‘toilet series’ (where he had to use his toilet paper to draw on, for all of his paper was filled with squares and rectangles), he dreamt of filling a large rectangle with small rectangles of width 2 and height 1 in varying ways.

Expert as he was in this material, he saw at a glance that he’ll need a computer to calculate the number of ways to fill the large rectangle whose dimensions were integer values, as well. Help him, so that his dream won’t turn into a nightmare!
Input

The input contains several test cases. Each test case is made up of two integer numbers: the height h and the width w of the large rectangle. Input is terminated by h=w=0. Otherwise, 1<=h,w<=11.
Output

For each test case, output the number of different ways the given rectangle can be filled with small rectangles of size 2 times 1. Assume the given large rectangle is oriented, i.e. count symmetrical tilings multiple times.
Sample Input

1 2
1 3
1 4
2 2
2 3
2 4
2 11
4 11
0 0
Sample Output

1
0
1
2
3
5
144
51205

翻译

给你一个n*m的棋盘，用1*2的多米诺骨牌去覆盖它，求方案数.

代码

分析

  多米诺骨牌覆盖的套路题. 以行为阶段，状压. 若第i位为1，则说明这是竖着的多米诺骨牌的上面一片，否则是其他情况. 转移时需要检查第i-1行与第i行的状态之间是否合法. 若$sta[i]and sta[i - 1]=0$并且$sta[i] | sta[i - 1]$中的每一段连续的0的个数都是偶数个，那么它们就是合法的. 第一个判断是为了让上端出现的多米诺骨牌下端也出现，第二个判断是为了使横着放的多米诺骨牌合法.
  一个坑点：千万在外层初始化每个状态中每一段连续的0的个数是不是偶数个，因为状态的位数与m有关. m对答案会有影响!
