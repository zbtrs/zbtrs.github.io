poj3696 The Luckiest number

发表于 2018-05-08 | 分类于 算法竞赛进阶指南

英文题面

Description

Chinese people think of ‘8’ as the lucky digit. Bob also likes digit ‘8’. Moreover, Bob has his own lucky number L. Now he wants to construct his luckiest number which is the minimum among all positive integers that are a multiple of L and consist of only digit ‘8’.

Input

The input consists of multiple test cases. Each test case contains exactly one line containing L(1 ≤ L ≤ 2,000,000,000).

The last test case is followed by a line containing a zero.

Output

For each test case, print a line containing the test case number( beginning with 1) followed by a integer which is the length of Bob’s luckiest number. If Bob can’t construct his luckiest number, print a zero.

Sample Input

8
11
16
0
Sample Output

Case 1: 1
Case 2: 2
Case 3: 0
Source

2008 Asia Hefei Regional Contest Online by USTC

翻译

类似88888这样的数，由n个8组成，求最少的n使得这个数是L的倍数.

提示

  写出式子，化简得到一个简单的同余式.

代码

分析

  先写出式子： $L | 8(10^x - 1) / 9$,将9乘过去: $9L | 8(10^x - 1)$.
  接下来怎么化简？不知道. 现在这个式子也无法求解，因为x的取值范围太大. 我们需要做的是将x的取值范围缩小. 需要先知道一个结论：
  若$a^x \% p == 1$,则x的最小取值一定是$φ(p)$的约数.证明如下：
  $φ(p) = kx + r$,$a^{kx + r} \% p == 1$则$a^{kx} * a^r \%p == 1$,而$a^{kx} \% p == 1$,所以$a^r \% p == 1$.若$r > 0$,$r$完全可以变成0来使得$x$更小，所以$r = 0$,成立.
  原式子中有$10^x - 1$，若将$-1$移到同余式的右边，则刚好可以用上这个结论.关键是如何消掉式子中的8.
  还有一个结论：互质的两个数不影响整除性，可以忽视其中一个数,可$9L$与$8$不一定互质，怎么办？ 左右两边同时除以$(9L,8)$即可，令$d = (9L,8)$,左边的$9L / d$与右边的$8 / d$互质，可以将$8 / d$拿走. 原式子变成了$(9L / d) | (10^x - 1)$. 可以利用上面的结论了.
  满心欢喜敲完代码交上去，发现WA了. Why? 因为对$9L / d$取模，模数可能大于int存储范围, 而快速幂可能就是两个大于int存储范围的数相乘，会爆long long,正确的方法是将快速幂中的乘法改成加法.

总结

  1.如何在整除式中越掉一个乘积？提公因数！
  2.一个扩展结论:模m意义下，与m互质的数对于整除性没有影响，也就是说：若$ac = bc (mod  m),(c,m) == 1—->a = c(mod  m)$
  3.模一个数等于1的次数最小值的取值范围.
