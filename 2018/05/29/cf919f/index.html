Codeforces 919 F.A Game With Numbers

发表于 2018-05-29 | 分类于 计划

###题目大意 两个人玩游戏，每个人手上各有8张牌，牌上的数字均为[0,4]之间的数，每个人在自己的回合选择自己的一张不为0的牌和对方的一张不为0的牌，分别记为a和b,令c = (a + b) % 5,用c替换掉a. 如果当前人的所有牌都变成了0，则其获胜，问谁能获胜. ###提示   较难解决的博弈论问题的常用方法，关键是如何表示状态？ ###代码

###分析   拿到一道博弈论题，先看看能不能找到规律，这道题中显然是找不到规律的. 如果不行，再来看能不能转化成有向图游戏利用sg函数求解. 在这道题中，如果一个人不能移动，那么这个人获胜，不符合用sg函数的条件. 那么只有用最后一种方法了：状态转移法.   如果当前局面的所有后继局面都是另一个先手的必胜局面，那么当前局面一定是必败局面. 如果后继局面出现了一个必败局面，那么当前局面一定是必胜局面. 如果将局面看作图上的点，当前局面与后继局面的关系看作边，那么原图就可以转换成一个DAG,利用拓扑排序即可求出所有的局面是必胜局面还是必败局面.   比较令人头疼的是状态数非常多，每个位置有5中选择，那么总共有$5^8$中选择，而一种局面是两个人的状态组合起来的，总状态数是$5^{16}$,存不下. 数的值域很小，那么可以关心每种值出现了多少次而不是每个位置是哪个值，利用隔板法(解方程)，可知状态数为$C(8 + 5 - 1,5 - 1)^2$,非常小. ###经验   一定要熟练掌握博弈论问题的几种通用解法. 值域与下标的转换也要记住.
