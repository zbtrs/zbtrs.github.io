bzoj2351 [BeiJing2011]Matrix

发表于 2018-05-04 | 分类于 算法竞赛进阶指南

###中文题面
Description
给定一个M行N列的01矩阵，以及Q个A行B列的01矩阵，你需要求出这Q个矩阵哪些在原矩阵中出现过。
所谓01矩阵，就是矩阵中所有元素不是0就是1。

Input
输入文件的第一行为M、N、A、B，参见题目描述。
接下来M行，每行N个字符，非0即1，描述原矩阵。
接下来一行为你要处理的询问数Q。
接下来Q个矩阵，一共Q*A行，每行B个字符，描述Q个01矩阵。

Output
你需要输出Q行，每行为0或者1，表示这个矩阵是否出现过，0表示没有出现过，1表示出现过。

Sample Input
3 3 2 2

111

000

111

3

11

00

11

11

00

11

Sample Output
1

0

1

HINT
对于100%的实际测试数据，M、N ≤ 1000，Q = 1000

对于40%的数据，A = 1。

对于80%的数据，A ≤ 10。

对于100%的数据，A ≤ 100。

###分析
  a,b的大小是固定的，预处理原矩阵中以每一个点为右下角的a*b的矩阵的hash值.放到map或set中查询. 自己写一个hash表也是可以的.
  涉及到二维hash,先对每一行上的点hash,求出以该点结尾的长度为a的序列的hash值，然后在列上hash,需要注意的是两次的模数不能相同！
