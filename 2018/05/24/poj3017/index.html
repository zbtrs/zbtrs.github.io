poj3017 Cut the Sequence

发表于 2018-05-24 | 分类于 算法竞赛进阶指南 |

英文题面

Description

Given an integer sequence { an } of length N, you are to cut the sequence into several parts every one of which is a consecutive subsequence of the original sequence. Every part must satisfy that the sum of the integers in the part is not greater than a given integer M. You are to find a cutting that minimizes the sum of the maximum integer of each part.

Input

The first line of input contains two integer N (0 < N ≤ 100 000), M. The following line contains N integers describes the integer sequence. Every integer in the sequence is between 0 and 1 000 000 inclusively.

Output

Output one integer which is the minimum sum of the maximum integer of each part. If no such cuttings exist, output −1.

Sample Input

8 17
2 2 2 8 1 8 2 1
Sample Output

12
Hint

Use 64-bit integer type to hold M.

Source

POJ Monthly—2006.09.29, zhucheng

翻译

  给定一个长度为N的序列A,要求把该序列分成若干段，在满足每段的和不超过M的前提下，让每段中所有数的最大值之和最小。计算这个值.

提示

  连续序列划分问题，用dp解，先写出状态转移方程，再来想用什么方法去优化。

代码

分析

  这是一道非常特殊的用单调队列优化的dp题. 先写出状态转移方程：
$f_i = min\{f_j + max\{a_k\}(j + 1≤k≤i)\}(Σa_k ≤ M(j+1≤k≤i))$
  枚举i和j的时间复杂度是$O(n^2)$的，需要优化掉其中一维的枚举. 一般是优化j的枚举. 观察状态转移方程，可以发现随着i的变大，j的取值范围的上界和下界都是单调变化的. 那么可以利用单调队列来优化.
  可以用单调队列优化的dp状态转移方程后面一般接一个多项式，这个多项式的每一项只与i或j有关.然而这个状态转移方程后面根本就算不上多项式. 那么需要特殊的方法去处理：
  1.找到最优解出现满足的情况,弄清楚这些情况是and关系还是or关系.
  2.单调队列维护满足上面条件的位置.
  3.用某种数据结构得到最优解.
  也就是说，用单调队列优化这类dp问题的转移和优化“后面接多项式”dp的转移是不一样的. 前者是维护限制条件，也就是可能是最优决策的位置，后者是直接维护最优决策.
  第一步是关键. 一般的方法是贪心地考虑有哪些情况可能成为最优解. 对于本题而言,有两种需要考虑的决策：
  1.$A_j$是[j,i]中的最大值，那么从$f_j$转移而来.
  2.从j+1开始，到i的和正好≤M的最小的j.
  为什么考虑上面两种转移呢？对于第一种转移可以这样贪心考虑：将序列分成两段s1和s2. 从$f_j$转移而来，说明将$A_j$放在了s1,$A_{j + 1}$到$A_i$都放在了s2. 若s1段的最大值不是$A_j$那么相当于s1的答案不变，而s2的答案变小了.若s1段的最大值是$A_j$,这种情况只会发生在边界. 因为按照这种转移方式，s1一定能被分成更小的段，使得有$A_k≥A_j$. 因为边界处的特殊性，所以要考虑第二种决策. 两种决策关系是or，也就是说只要满足其中的一条，就有可能成为最优决策.
  当YY出两种决策后，就要利用决策选择性来证明. 利用反证法，假设决策j两个条件都不满足, 因为$f_j$的单调性和max的单调性，$f_{j - 1} + max\{A_k\} (j≤k≤i) ≤ f_j + max\{A_k\}(j + 1≤k≤i)$. 那么我们完全可以选$f_{j - 1}$而不是$f_j$. 这个不等式是建立在两个条件都不满足的基础之上的，所以这两种条件至少要满足一个.
  然后将所有可能的情况看作图上的点，对于and关系的情况，在它们之间连上一条边，那么图中就会有若干个连通分量. 分别对这些连通分量转移.
  比如这两个限制条件的关系是or,那么就有两个连通分量，分开来考虑. 对于第二个决策，用一个单调指针维护从哪里转移即可. 对于第一个决策. 维护一个j递增，$A_j$递减的队列，用单调队列正好可以维护. 而$A_j$是j到i中最大的一个数，那么单调队列中的第i项的f值和第i+1项的a值即为一种可能的转移. 注意下标运算.
  因为用单调队列维护的是可能成为最优决策的位置，要用一个数据结构维护最优值. 这个数据结构要支持删除最优值、添加一个值、取出最优值这些操作. multiset是首选，当然两个优先队列也是可以的.
