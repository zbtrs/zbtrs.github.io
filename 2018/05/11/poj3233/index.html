poj3233 Matrix Power Series

发表于 2018-05-11 | 分类于 算法竞赛进阶指南

英文题面

Description

Given a n × n matrix A and a positive integer k, find the sum $S = A + A^2 + A^3 + … + A^k.$

Input

The input contains exactly one test case. The first line of input contains three positive integers n (n ≤ 30), k (k ≤ 109) and m (m < 104). Then follow n lines each containing n nonnegative integers below 32,768, giving A’s elements in row-major order.

Output

Output the elements of S modulo m in the same way as A is given.

Sample Input

2 2 4
0 1
1 1
Sample Output

1 2
2 3
Source

POJ Monthly—2007.06.03, Huang, Jinsong

翻译

求一个矩阵的等比数列和.

分析

  求等比数列可以利用公式，也可以分治. 在本题中采用分治的方式. 一些问题需要注意：
  1.尽量减少不必要的取模.
  2.能不用long long就尽量不用long long.
  3.读入了数据后要取模.
  有了这3个优化，程序就能在规定时间内跑完啦. 如果取模只涉及到加减运算，那么就可以将取模操作变成减法操作. 用long long会非常耗费时间.
  求等比数列时要对n分奇偶性讨论. 数学老师CKM曾在课堂上讲过：对于要奇偶分类讨论的题，先求出偶数的情况，那么奇数的情况就能通过-1从偶数的情况上推导过来. 正好可以应用在本题上.
