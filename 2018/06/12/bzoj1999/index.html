bzoj1999 [Noip2007]Core树网的核

发表于 2018-06-12 | 分类于 算法竞赛进阶指南 |

中文题面

Description
设T=(V, E, W) 是一个无圈且连通的无向图（也称为无根树），每条边带有正整数的权，我们称T为树网（treenetwork），其中V, E分别表示结点与边的集合，W表示各边长度的集合，并设T有n个结点。 路径：树网中任何两结点a,b都存在唯一的一条简单路径，用d(a,b)表示以a,b为端点的路径的长度，它是该路径上各边长度之和。我们称d(a,b)为a,b两结点间的距离。 一点v到一条路径P的距离为该点与P上的最近的结点的距离： d(v，P)=min{d(v，u)，u为路径P上的结点}。 树网的直径：树网中最长的路径称为树网的直径。对于给定的树网T，直径不一定是唯一的，但可以证明：各直径的中点（不一定恰好是某个结点，可能在某条边的内部）是唯一的，我们称该点为树网的中心。 偏心距ECC(F)：树网T中距路径F最远的结点到路径F的距离，即 。 任务：对于给定的树网T=(V, E,W)和非负整数s，求一个路径F，它是某直径上的一段路径（该路径两端均为树网中的结点），其长度不超过s（可以等于s），使偏心距ECC(F)最小。我们称这个路径为树网T=(V,E,W)的核（Core）。必要时，F可以退化为某个结点。一般来说，在上述定义下，核不一定只有一个，但最小偏心距是唯一的。 下面的图给出了树网的一个实例。图中，A-B与A-C是两条直径，长度均为20。点W是树网的中心，EF边的长度为5。如果指定s=11，则树网的核为路径DEFG（也可以取为路径DEF），偏心距为8。如果指定s=0（或s=1、s=2），则树网的核为结点F，偏心距为12。
Input
包含n行： 第1行，两个正整数n和s，中间用一个空格隔开。其中n为树网结点的个数，s为树网的核的长度的上界。设结点编号依次为1, 2, …, n。 从第2行到第n行，每行给出3个用空格隔开的正整数，依次表示每一条边的两个端点编号和长度。例如，“2 4 7”表示连接结点2与4的边的长度为7。 所给的数据都是正确的，不必检验。
Output
只有一个非负整数，为指定意义下的最小偏心距。
Sample Input
5 2

1 2 5

2 3 2

2 4 4

2 5 3

Sample Output
5
HINT
对于70%的数据，n<=200000
对于100%的数据：n<=500000, s<2^31, 所有权值<500

代码

分析

  原题$O(n^3)$就可以水过，不知道还有$O(n)$的算法，今天看了一看，真tm厉害.
  做这道题需要大胆猜测一个性质：在一个直径上求得的答案等于在所有直径上求得的答案. 假设直径的两个端点为p,q，可以枚举核，在直径的每个点上都做一次dfs,求出最远距离，取一下max就得到了偏心距,时间复杂度$O(n^3)$.
  继续分析，答案取决于核外的点到核上的任意一个点最小距离. 这是针对核外的某个点而言的. 为了使得答案尽量小，肯定是核越长越好，所以枚举核只需要枚举一个端点即可.时间复杂度$O(n^2)$
  题目要求的实质是让最大值最小，很容易想到二分. 二分偏心距的最大值，怎么判断可行性呢？判断的过程肯定是越简单越好，如果还是一个一个点去求偏心距，那么就不需要二分了. 需要利用到直径的性质. 假设核靠近p的一端是u,因为(p,q)是直径，所以(p,u)上延伸出去的分支到u的距离一定不大于p到u的距离. 否则p就不可能是直径的端点. 那么只需要枚举u，v的位置可以通过维护单调指针得到，判断两个端点到u,v的距离的最大值即可.时间复杂度$O(nlogsum)$
  还没完，根据上述分析，可以知道答案一定是以下几种情况：
  1.p,q到u,v的距离.
  2.(p,q)上的点能延伸到的最远距离.
  枚举u,v的位置是单调递增的，那么可以用单调队列维护.时间复杂度$O(n)$.
  还没完，之前维护的是(p,q)上的点能延伸到的最远距离. 事实上，根据(p,u)上延伸出去的分支到u的距离一定不大于p到u的距离这一性质，(p,q)之外的点能延伸到的最远距离不可能成为答案，因为最后取max,可以将直径上所有的点能延伸到的最远距离归为第二种情况. 就不需要用单调队列维护了. 只需要求直径的两个端点到核的两个端点的距离即可.
  其实我觉得正常人应该能一下子想到最后一步，写出答案的表达式即可. 只是直径的最长性可能会有点困难想到.
