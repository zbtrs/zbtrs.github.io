bzoj1237 [SCOI2008]配对

发表于 2018-06-04 | 分类于 计划

###中文题面 Description 你有n 个整数Ai和n 个整数Bi。你需要把它们配对，即每个Ai恰好对应一 个Bp[i]。要求所有配对的整数差的绝对值之和尽量小，但不允许两个相同的数配 对。例如A={5,6,8}，B={5,7,8}，则最优配对方案是5配8, 6配5, 8配7，配对整数 的差的绝对值分别为2, 2, 1，和为5。注意，5配5，6配7，8配8是不允许的，因 为相同的数不许配对。

Input 第一行为一个正整数n，接下来是n 行，每行两个整数Ai和Bi，保证所有 Ai各不相同，Bi也各不相同。

Output 输出一个整数，即配对整数的差的绝对值之和的最小值。如果无法配对，输 出-1。

Sample Input 3

3 65

45 10

60 25 Sample Output 32 HINT 1 <= n <= 10^5，Ai和Bi均为1到10^6之间的整数。 ###提示   若没有限制该怎么做？为什么要这么做？这种做法满足什么原则？ ###代码

###分析   如果没有限制，将a,b两个序列的数分别从小到大排序后一一配对即可. 这体现了一个最优性原则：每个数都与离它尽可能近的数配对.   如果有限制，那么就会有数与离他不是最近的数配对. 比如a,b两个数与对应位置上的数冲突了，那么a可以匹配b对应位置上的数，b可以匹配a对应位置上的数,此时每个数与配对的数的最长距离为1.如果a,b,c三个数都与对应位置上的数冲突了,此时不管怎么交叉着匹配，最长距离的最小值一定为2. 如果a,b,c,d四个数都冲突了，那么可以拆分成两个a,b冲突的情况，最长距离为1. 由此可以得到一个结论：最优解中，每个数与匹配位置的距离≤2.   由此dp. 令$f_{i}$表示前i个数的答案，分类讨论4种转移，从$f_{i - 2}$或$f_{i - 3}$转移过来.
