poj1390 Blocks

发表于 2018-06-02 | 分类于 算法竞赛进阶指南

英文题面

Description

Some of you may have played a game called ‘Blocks’. There are n blocks in a row, each box has a color. Here is an example: Gold, Silver, Silver, Silver, Silver, Bronze, Bronze, Bronze, Gold.
The corresponding picture will be as shown below:

Figure 1

If some adjacent boxes are all of the same color, and both the box to its left(if it exists) and its right(if it exists) are of some other color, we call it a ‘box segment’. There are 4 box segments. That is: gold, silver, bronze, gold. There are 1, 4, 3, 1 box(es) in the segments respectively.

Every time, you can click a box, then the whole segment containing that box DISAPPEARS. If that segment is composed of k boxes, you will get kk points. for example, if you click on a silver box, the silver segment disappears, you got 44=16 points.

Now let’s look at the picture below:

Figure 2

The first one is OPTIMAL.

Find the highest score you can get, given an initial state of this game.
Input

The first line contains the number of tests t(1<=t<=15). Each case contains two lines. The first line contains an integer n(1<=n<=200), the number of boxes. The second line contains n integers, representing the colors of each box. The integers are in the range 1~n.
Output

For each test case, print the case number and the highest possible score.
Sample Input

2
9
1 2 2 2 2 3 3 3 1
1
1
Sample Output

Case 1: 29
Case 2: 1
Source

Liu Rujia@POJ

翻译

  n个方块，每种方块都有一种颜色，每次可以选择消掉一个方块，与方块相连的所有同色方块都会被消掉，代价为消掉的方块数的平方，求最大代价.

提示

  消数，求最值，区间dp.

代码

分析

  这道题非常容易想到区间dp. 按照区间dp的常规套路来，令$f_{i,j}$表示区间[i,j]的最大代价. 有些情况是转移不了的. 比如多个不相邻相同颜色的色块合并，转移只能枚举k,最多只能处理3个相同颜色的色块的合并.
  区间外会影响区间内，端点色块可能会和区间外的相同色块合并，二维状态只能维护区间内部的情况，如果要处理区间外对区间内的影响，就必须再多加一维. 令$f_{i,j,k}$表示区间[i,j],右端点j与其右边的k个相同色块合并的最大代价. 转移的时候考虑右端点色块，要么只和区间外的色块合并，要么和区间内的色块合并.

总结

  区间dp的常规状态表示只能处理区间内部的情况，对于区间外部的情况需要多加一维. 只需要考虑左端点or右端点向外扩张的情况即可.
